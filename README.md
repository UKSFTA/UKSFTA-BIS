# UKSFTA-BIS

**ArmA File Format Library for UKSF Taskforce Alpha development pipeline.**

A centralized, cross-platform library providing high-performance reading and manipulation capabilities for common Bohemia Interactive ArmA file formats.

## 🛠 Features

- **Format Versatility**: Native support for P3D (ODOL/MLOD), PBO, PAA/PAC, RTM, SQFC, and WRP.
- **Modernized Infrastructure**: Targets .NET 10.0 for maximum performance and cross-platform compatibility.
- **Developer Optimized**: Standardized build and development workflow for streamlined integration.

## 🔧 Technical Logic

- **Language**: C# / .NET 10.0
- **Build System**: Automated via standardized `build.sh` and `dev.sh` scripts.
